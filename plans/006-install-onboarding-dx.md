### Step 2: Zed troubleshooting playbook

Expand `zed-extension/README.md` § Troubleshooting:

**Panels look flat / same color as editor**

- Confirm `surface.background` ≠ `background` in `zed: open log` or inspect `themes/tokyonight-theme.json`
- Re-run `bun run install:zed-dev` after `bun run build`
- Compare with `zed/golden/tokyonight.json`

**Wrong syntax colors (e.g. variables show `M.base_16...`)**

- Run `bun run build` — indicates stale extension
- `zed: reload` or restart Zed

**Theme not in picker**

- Dev extension path must be `zed-extension/` not repo root
- Check `extension.toml` id = `nvchad-themes`

**Prerequisites**

- Bun for dev workflow; OR download [dist zip](../dist/) without cloning
